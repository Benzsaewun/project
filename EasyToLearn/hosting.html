    • Introduction

    Hosting refers to the act of making a website or application accessible on the internet. This is typically done by renting server space from a hosting provider, who will then store the website or application on their servers and make it available to users via the internet.

    Hosting services can include a variety of features such as domain registration, email hosting, and database management, among others.

    Free Web Hosting

    Free web hosting is a service that allows individuals and organizations to host their website or application on the internet at no cost.

    However, free web hosting usually comes with some limitations such as less storage, less bandwidth, ads displaying on your website, and less control over the server environment.

    Step to Hosting in firebase

    you must have vscode and nodejs

    Go to website firebase > Go to console > Add project > Create name project > Enable google analytics > select an account > create project

    console firebase page > menu Build > Hosting > Get started > copy npm and paste to command prompt > click enter to install firebase CLI

    you can check version firebase. > write firebase -V > write firebase login > collect CLI usage and error information write Y > choose an account

    vs code > menu view > Terminal > write firebase init > select Hosting: Configure files for firebase Hosting > use an existing project > select firebase project > single-page write Y if not write N > deploys with Github write N >

    firebase will crate folder: public > delete index.html in folder public and move file project into folder public (index.html = main page, must have index.html file) > write firebase deploy in terminal vs code > hosting URL = your project

    Step to Hosting in Google App Script

    Go to website Google App Script > Start Scripting > New project > write function in code.gs

    Code.gs

    function doGet(e) {
      return HtmlService.createTemplateFromFile('index').evaluate()
      .addMetaTag('viewport','width=device-width , initial-scale=1')
      .setXFrameOptionsMode(HtmlService.XFrameOptionsMode.ALLOWALL)
    }

    function include(filename) {
      return HtmlService.createTemplateFromFile(filename).getContent();
    }
