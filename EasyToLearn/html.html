    What is HTML?
    HTML (Hypertext Markup Language)
    HTML is a markup language used to create the structure of a webpage.

    <html>
    It is used to start and end the HTML document.

    <head>
    It is used to define the heading part of the page. The contents of this tag arenâ€™t displayed in the webpage, but it helps the web browser to get inner information of the webpage.

    <title>
    It is used to define the title of the webpage.

    <body>
    It is used to define the body parts of the hypertext document. It contains all the information of the webpage. The contents of this tag are displayed in the webpage.

    Shortcut keys for building instructure HTML is ! and html:5

    HTML Document

    <!DOCTYPE html>
    <html>
    <head>
    <title>Page Title</title>
    </head>
    <body>

    <h1>My First Heading</h1>
    <p>My first paragraph.</p>

    </body>
    </html>

    1.HTML Headings

    HTML headings are defined with the <h1> to <h6> tags.
    <h1> defines the most important heading.
    <h6> defines the least important heading.
